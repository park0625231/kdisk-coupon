쿠폰 등록 단계별 가이드

쉽고 빠르게 쿠폰을 등록하세요

1

쿠폰 코드 복사

위의 쿠폰 코드 중 하나를 선택하여 '복사' 버튼을 클릭합니다. 클립보드에 자동으로 복사됩니다.

예: kkkkk-8416 복사
2

등록 페이지 접속

'쿠폰 등록 페이지로 이동' 버튼을 클릭하여 케이디스크 공식 쿠폰 등록 페이지로 이동합니다.

3

로그인

케이디스크 계정으로 로그인합니다. 계정이 없다면 회원가입을 진행하세요.

4

쿠폰 코드 입력

쿠폰 입력 필드에 복사한 코드를 붙여넣기(Ctrl+V 또는 Cmd+V)합니다.

입력 예: kkkkk-8416
5

등록 완료

'확인' 또는 '등록' 버튼을 클릭하면 쿠폰이 등록됩니다. 프리미엄 기능을 즉시 이용할 수 있습니다.

6

다음 쿠폰 등록 (선택)

첫 번째 쿠폰이 만료되기 전에 다른 쿠폰을 등록하여 프리미엄 기능을 계속 유지할 수 있습니다.

예: kkkkk-8417, kkkkk-8418 순서로 등록
